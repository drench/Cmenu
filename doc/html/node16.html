Internal Functions

These are the internal functions of the module; these should not be called by user scripts.
menu_advice
displays a short message at the foot of the screen
menu_button_bar
creates a button bar
menu_create_pane
creates the menu display
menu_draw_active
draws the active menu option
menu_draw_button
draws a button
menu_draw_inlay
draws the backdrop
menu_draw_line
draws an inactive menu option
menu_draw_pane
draws the menu pane
menu_draw_window
draws the menu window
menu_edit
edit a field
menu_hot_button
report which button is hot
menu_key_seq
get input from the keyboard
menu_navigate
navigate through a menu
menu_preferences
load modules preferences from file
menu_redraw_backdrop
redraw the backdrop
menu_refresh
redraw the whole screen especially after resizing a terminal window
