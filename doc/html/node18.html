tic file

Provided with the module is a tic file for VT100 emulation on a Wyse 60 terminal. This has some of the function keys defined correctly. You will need to define function strings if you want to use the 16 F keys. Page and some of the character keys are mapped [NB the DEL key sends line-control info and cannot be used on mine - maybe yours is OK - also the numeric keypad can misbehave]
