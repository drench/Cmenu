User Functions

These functions are designed to be called by user scripts.
menu_button_set
enables or disables buttons; sets text on button
menu_display
call to activate a menu and return a selection
menu_init
initialise a new menu structure
menu_initialise
initialise the module; called at the beginning of a script once only
menu_item
define a menu option
menu_popup
popup a display while processing is occurring
menu_show
splash data to the screen
menu_terminate
terminate all Curse and menuing structures; called on exit from script
